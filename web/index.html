How it works:
  • Save Configuration: Saves settings without restarting (changes apply gradually)
  • Save & Restart Service: Immediate restart to test new settings
  • Force Hotspot: Keeps hotspot active regardless of network availability
  • Without force mode: Primary network (4 attempts) → Backup network (4 attempts) → Hotspot
  • Device automatically switches back when networks become available
